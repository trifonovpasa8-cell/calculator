# SmartCalculator --- Керівництво користувача

SmartCalculator --- це зручний, багатофункціональний калькулятор,
створений для швидких і точних обчислень. Додаток поєднує простоту
використання зі зручним інтерфейсом та розширеними можливостями.

## Основні можливості

-   **Стандартні арифметичні операції**
    -   Додавання, віднімання, множення, ділення
-   **Розширені математичні функції**
    -   Степені, корені, відсотки, факторіал
-   **Підтримка інженерних функцій**
    -   Синус, косинус, тангенс, логарифми
-   **Історія обчислень**
    -   Перегляд попередніх дій
    -   Можливість повторного використання результатів
-   **Гнучкі налаштування**
    -   Темна/світла тема
    -   Налаштування точності
-   **Зручний інтерфейс**
    -   Інтуїтивні кнопки
    -   Адаптивний дизайн для різних пристроїв

##  Початок роботи

1.  **Встановіть додаток**\
    Завантажте SmartCalculator з офіційного джерела або магазину
    застосунків.

2.  **Запустіть додаток**\
    На головному екрані ви побачите стандартну панель кнопок та поле
    введення.
